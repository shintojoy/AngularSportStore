File Storage

File system storage is chargable. Follow the below steps to create a file system and mount it in Ubundu.

  • Go to Storage -> Filestorage ->Create Instance
  • Provide an instance id (any name)
  • Instance tier is standard and select a region and all its zones
  • Provide a file share name and create it
  • Create 2 ubundu VM's and mount the above created file system using the procedure listed below.
    • Get the remote access of the ubundu VMs using putty or directly from from google console by clicking the SSH
    • sudo apt update
    • sudo apt install nfs-common -y , This command installs the nfs file system in ubundu
    • sudo mkdir /memo , This creates a folder named memo in VM, for the second VM the name can be nemo.
    • Run the command sudo vim /etc/fstab this opens up an editor
    • Copy the below path into the file as the last line.

      10.146.31.146:/web /memo nfs auto,nofail,noatime,nolock,intr,tcp,actimeo=1800 0 0

      IP address should be copied from the file store created in google console :wq to save the file enter the exit the file

    • sudo mount -a to mount the file storage
    • df -h this is to see the mounted files
  • Create a folder/directory using mkdir in the newly mounted folder nemo/memo
  • Create a file using the command sudo touch note.txt in the newly created folder
  • Use ll to list and check whether both vm having the same list of files/folders, this should be from nemo/memo.
